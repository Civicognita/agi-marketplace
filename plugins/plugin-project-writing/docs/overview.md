# Writing Project Type

## Overview

Project type for literature and long-form writing — novels, screenplays, research documents, and essays. Marked as hostable; when hosted it is served by the **Reader** MApp, which presents content in a book-style layout.

## Details

| Property | Value |
|----------|-------|
| ID | `writing` |
| Category | literature |
| Hostable | Yes |
| Default mode | production |
| Viewer MApp | plugin-reader-literature |

## Built-in Tools

| Tool | Description |
|------|-------------|
| Word Count | Count words across all `.md` and `.txt` files |
| Export | Export the project for publishing |
| Outline | View document outline |

## Usage

1. Create a new project and select **Writing Project** as the project type.
2. Write content as Markdown (`.md`) or plain text (`.txt`) files.
3. Open the **Hosting** panel to enable the Reader MApp and get a `<hostname>.ai.on` URL for sharing.

## Notes

- The Reader MApp must be installed from the Plugin Marketplace to enable hosted reading.
- Stack plugins for novel, screenplay, and research writing add scaffolding and structural templates on top of this project type.
